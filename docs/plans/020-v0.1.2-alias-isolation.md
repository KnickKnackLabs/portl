# v0.1.2 Alias Isolation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use
> superpowers:subagent-driven-development (recommended) or
> superpowers:executing-plans to implement this plan task-by-task.
> Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace `aliases.sqlite` with `aliases.json` + `fd-lock`,
add a one-shot migration path from the SQLite file, and ship as
v0.1.2 so we can test the rusqlite-as-macOS-crash-cause hypothesis
in isolation (see
[`../specs/160-v0.1.2-alias-isolation.md`](../specs/160-v0.1.2-alias-isolation.md)).

**Architecture:** A new `AliasStore` backed by JSON with
`fd-lock`-serialised writers. A tiny `migrate_from_sqlite()` runs
once at startup if legacy `aliases.sqlite` exists. `rusqlite` stays
in `portl-cli` only; `portl-agent` does not depend on it.

**Tech Stack:** `serde_json`, `fd-lock`, `tempfile` (for atomic
rename), `rusqlite` (kept for the migration path only, dropped in
v0.2).

```rust
// crates/portl-cli/src/alias_store_json.rs
use anyhow::{Context, Result};
use fd_lock::RwLock;
use serde::{Deserialize, Serialize};
use std::collections::BTreeMap;
use std::fs::{File, OpenOptions};
use std::io::{Read, Seek, SeekFrom, Write};
use std::path::{Path, PathBuf};

#[derive(Serialize, Deserialize, Clone, Debug)]
pub struct AliasRecord {
    pub endpoint_id: String,
    pub ticket: String,
    #[serde(default = "now_rfc3339")]
    pub created_at: String,
    #[serde(default = "now_rfc3339")]
    pub updated_at: String,
}

#[derive(Serialize, Deserialize, Default)]
struct Root { version: u32, aliases: BTreeMap<String, AliasRecord> }

pub struct AliasStore { path: PathBuf, lock_path: PathBuf }

impl AliasStore {
    pub fn open(dir: &Path) -> Result<Self> {
        std::fs::create_dir_all(dir)?;
        Ok(Self {
            path: dir.join("aliases.json"),
            lock_path: dir.join("aliases.json.lock"),
        })
    }

    pub fn upsert(&self, name: &str, rec: AliasRecord) -> Result<()> {
        self.with_write_lock(|root| {
            root.version = 1;
            root.aliases.insert(name.to_owned(), rec);
            Ok(())
        })
    }

    pub fn get(&self, name: &str) -> Result<Option<AliasRecord>> {
        let root = self.read()?;
        Ok(root.aliases.get(name).cloned())
    }

    pub fn list(&self) -> Result<Vec<(String, AliasRecord)>> {
        let root = self.read()?;
        Ok(root.aliases.into_iter().collect())
    }

    fn read(&self) -> Result<Root> {
        if !self.path.exists() { return Ok(Root::default()); }
        let f = File::open(&self.path)?;
        let mut lock = RwLock::new(f);
        let guard = lock.read()?;
        let mut s = String::new();
        let mut f2: &File = &*guard;
        f2.read_to_string(&mut s)?;
        drop(guard);
        if s.trim().is_empty() { return Ok(Root::default()); }
        serde_json::from_str(&s).context("parse aliases.json")
    }

    fn with_write_lock<F: FnOnce(&mut Root) -> Result<()>>(&self, f: F) -> Result<()> {
        let lock_file = OpenOptions::new().read(true).write(true).create(true)
            .open(&self.lock_path)?;
        let mut lock = RwLock::new(lock_file);
        let _g = lock.write()?;
        let mut root = self.read()?;
        f(&mut root)?;
        let tmp = self.path.with_extension("json.tmp");
        let mut out = File::create(&tmp)?;
        out.write_all(serde_json::to_vec_pretty(&root)?.as_slice())?;
        out.sync_all()?;
        std::fs::rename(&tmp, &self.path)?;
        Ok(())
    }
}

fn now_rfc3339() -> String {
    chrono::Utc::now().to_rfc3339()
}

impl AliasRecord {
    #[cfg(test)]
    pub fn test(endpoint_id: String) -> Self {
        Self { endpoint_id, ticket: "test-ticket".into(),
               created_at: now_rfc3339(), updated_at: now_rfc3339() }
    }
}
```

```rust
// crates/portl-cli/src/alias_migrate.rs
use anyhow::{anyhow, Context, Result};
use rusqlite::Connection;
use std::path::Path;
use crate::alias_store_json::{AliasStore, AliasRecord};

pub struct MigrationResult { pub migrated: bool, pub alias_count: usize }

pub fn migrate_if_needed(dir: &Path) -> Result<MigrationResult> {
    let sqlite = dir.join("aliases.sqlite");
    let json = dir.join("aliases.json");
    match (sqlite.exists(), json.exists()) {
        (false, _) => Ok(MigrationResult { migrated: false, alias_count: 0 }),
        (true, true) => Err(anyhow!(
            "both aliases.sqlite and aliases.json exist in {}; refusing to migrate. \
             Inspect and remove one.", dir.display())),
        (true, false) => {
            let conn = Connection::open_with_flags(
                &sqlite,
                rusqlite::OpenFlags::SQLITE_OPEN_READ_ONLY,
            )?;
            let mut stmt = conn.prepare("SELECT alias, endpoint_id, ticket FROM aliases")?;
            let rows = stmt.query_map([], |r| Ok((
                r.get::<_, String>(0)?,
                r.get::<_, String>(1)?,
                r.get::<_, String>(2)?,
            )))?.collect::<rusqlite::Result<Vec<_>>>()?;
            let store = AliasStore::open(dir)?;
            for (name, eid, ticket) in &rows {
                store.upsert(name, AliasRecord {
                    endpoint_id: eid.clone(),
                    ticket: ticket.clone(),
                    created_at: crate::alias_store_json::now_rfc3339(),
                    updated_at: crate::alias_store_json::now_rfc3339(),
                })?;
            }
            drop(stmt); drop(conn);
            let ts = chrono::Utc::now().format("%Y%m%dT%H%M%SZ");
            std::fs::rename(&sqlite, dir.join(format!("aliases.sqlite.migrated-{ts}")))?;
            Ok(MigrationResult { migrated: true, alias_count: rows.len() })
        }
    }
}
```

Expose `now_rfc3339` as `pub(crate)` in `alias_store_json.rs`.
